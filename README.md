# Human Discomfort Indicator with Blynk IoT

This project monitors temperature and humidity levels to determine human comfort or discomfort conditions.  
The system displays the result on an LCD and sends real-time environmental data to the Blynk IoT platform for remote monitoring.

---

## Features

- Real-time temperature and humidity monitoring
- Comfort / Discomfort detection
- LCD display output
- Remote monitoring using Blynk IoT
- Arduino based embedded system

---

## Components Used

- ESP8266
- DHT11 Temperature and Humidity Sensor
- I2C LCD Display
- Breadboard
- Jumper wires

---

## Circuit Connections

**DHT11 Sensor**

- VCC → 3.3V / 5V  
- GND → GND  
- Data → D2 (GPIO4)

**I2C LCD**

- VCC → 5V  
- GND → GND  
- SDA → D2 / GPIO4  
- SCL → D1 / GPIO5
